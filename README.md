# Berry Coin Official Website

A modern, professional, single-page, fully responsive React website for Berry Coin - a next-generation decentralized digital asset built to bridge blockchain innovation with real-world value.

## Features

- **Modern Design**: Clean, professional aesthetics with Inter font
- **Fully Responsive**: Mobile-first design that works perfectly on all devices
- **Dark/Light Mode**: Toggle between themes with system preference detection
- **Smooth Animations**: Floating elements, hover effects, and transitions
- **Comprehensive Sections**: 
  - Hero section with call-to-action
  - Vision and problem/solution overview
  - 6-component ecosystem breakdown
  - Detailed tokenomics and allocation
  - 7-phase development roadmap
  - Security and compliance measures
  - Professional footer with disclaimers

## Technology Stack

- **React 18**: Modern React with hooks
- **Tailwind CSS**: Utility-first CSS framework
- **Lucide React**: Beautiful, customizable icons
- **PostCSS**: CSS processing
- **Inter Font**: Professional typography

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Build for production:
```bash
npm run build
```

## Color Scheme

- **Primary**: Blues (#3b82f6) for authority and trust
- **Accent**: Berry reds/purples (#ec4899) for calls-to-action
- **Background**: White/dark gray with gradient effects
- **Text**: Balanced contrast for readability

## Sections Overview

1. **Header & Navigation**: Fixed header with smooth scroll navigation
2. **Hero**: Main value proposition with animated elements
3. **Vision**: Problem identification and core solutions
4. **Ecosystem**: 6 interconnected components (DEX, Staking, NFT, Pay, DAO, RWA)
5. **Tokenomics**: Supply details, allocation, and use cases
6. **Roadmap**: 7-phase development timeline
7. **Security**: Trust indicators and compliance measures
8. **Footer**: Social links, disclaimers, and legal information

## Deployment

This website is ready for deployment to any static hosting service like Netlify, Vercel, or GitHub Pages.
